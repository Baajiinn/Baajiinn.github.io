## Objectifs et critères d’évaluation

# Fichiers
-  structure de votre dossier et sous-dossiers
- 0 fichiers inutiles
- noms, poids et formats de fichiers adaptés au Web

# HTML
- respecter norme W3C

# CSS
- lisibilité et organisation
- Responsive ( largeur : 320px - 2560px )
- Eviter les vides ( travailler sur plusieurs colonnes en desktop mais une seul en mobile )

# Taille du site
- Si hauteur > 2400px : informations trop espacées ou mal agencées !
- Largeur max : 2560px

# Contenu
- Quantité d'informations avec qualité

# Othographe 
- Faire un copier coller dans un correcteur d'orthoagaraphe.

# Pas de navigation

# Utilisation de JS ou jQuery
- Utiliser au moins 2 fonctionnalitées JS ou jQuery dont une doit être un **PLUGIN** trouvé en **LIGNE** 
- Eviter les contenus cachés qui se déploient au clic
- Quelques idées :
    -- 1 **effet lightbox** sur les images
    -- 1 **effet lazyload** lié au scroll
    -- des images qui se chargent avec une animation inhabituelle ( **effet escalier** )
    -- 1 **effet parallaxe**
    -- 1 bouton pour passer en **mode nocturne**
    -- 1 bouton pour **imprimer le site**
    -- etc.

# Apparance
- doit transpirer le professionnalisme et l'informatique. ( le visiteur doit comprendre que c'est un CV )
